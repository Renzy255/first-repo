PSEUDO ELEMENTS

A CSS pseudo-element is a special keyword in CSS that allows you to style specific parts of an element's content without having to add more HTML
code. Think of it like a way to apply extra styles to certain parts of an element, even though
those parts donâ€™t actually exist as separate elements in the HTML.



The p::selection property in CSS is used to change the appearance of the portion of text within a (paragraph) element when a user highlights or selects it. Normally, when you select text on a webpage (by clicking and dragging your mouse over it), the default style is usually a blue background with white text. However, with the ::selection pseudo-element, you can customize this look, allowing you to change the background color, text color, and even other styling aspects of the selected text.

The ::marker pseudo-element in CSS is used to style the markers (like bullets or numbers) of list items in ordered (ol) and unordered (ul) lists. Normally, these markers are simple dots, squares, or numbers, depending on the type of list. With the ::marker pseudo-element, you can customize their appearance, changing their color, size, font, and other properties.
