CSS OVERFLOW

OVERFLOW: VISIBLE

This is the default setting for the overflow property, meaning any content that spills over the edges of its container will be shown outside the container's box. This can be useful for designs where overflow content is intentional and should remain visible without restrictions.

OVERFLOW: HIDDEN

This value hides any content that extends beyond the bounds of its container, effectively clipping off the overflowed parts and making them invisible. Its useful for creating clean, contained layouts where you don't want any content to be displayed outside the designated area.

OVERFLOW: CLIP

The overflow: clip; property hides any content that extends beyond the containers border, ensuring that overflowed content is clipped and not visible outside the container. The overflow-clip-margin: 12px; property adds a 12-pixel margin inside the containerâ€™s border, creating a buffer zone before the content is clipped, which helps fine-tune the clipping effect.

OVERFLOW: SCROLL

When you set overflow to scroll, scrollbars will always appear on the container, regardless of whether the content actually overflows or not. This ensures that users can always scroll to see the content, but can result in unnecessary scrollbars if the content fits within the container.

OVERFLOW: AUTO

The auto value adds scrollbars to the container only when the content exceeds the containers dimensions, otherwise, no scrollbars are shown. This makes it a flexible option for handling overflow, as it only introduces scrollbars when necessary.
